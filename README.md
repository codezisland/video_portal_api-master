Installation
	npm install


To run:
	1 - run mongodb 
		"C:\Program Files\MongoDB\Server\3.4\bin\mongod.exe"

	2- npm start
	
To test
	npm test
	
	to run test you need to configure karma-cli or updated package.json test variable
	
	
*** Front-end frameworks and libraries ***
	AngularJS 1.6, Bootstrap 3.7, Jquery
	
*** Test tools
	Karma, Jasmine
